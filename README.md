# Vehicle Companies Research

אתר מחקרי בעברית על עשר חברות רכב:
Toyota, Hyundai, Kia, Škoda, Mitsubishi, Mazda, Honda, Suzuki, Peugeot, Volvo.

## איך מפעילים בגיטהאב Pages
1. יוצרים מאגר חדש בשם `vehicle-companies-research`.
2. מעלים את כל הקבצים שבתיקייה הזו.
3. נכנסים ל־Settings → Pages.
4. בוחרים Deploy from branch.
5. בוחרים את הענף `main` ואת התיקייה `/root`.
6. שומרים וממתינים לקישור האתר.

## מבנה
- `index.html` — דף ראשי
- `companies/` — דף נפרד לכל חברה
- `assets/` — כרטיסי SVG מקוריים לכל חברה
- `style.css` — עיצוב
- `script.js` — חיפוש בדף הראשי
